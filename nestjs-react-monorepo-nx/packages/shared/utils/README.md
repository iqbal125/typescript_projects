# @org/shared-utils

Shared utility package used by workspace apps and packages.

## Essentials

Run from workspace root:

```bash
pnpm nx build utils
pnpm nx typecheck utils
```

Use in code:

```ts
import { someUtil } from '@org/shared-utils';
```
